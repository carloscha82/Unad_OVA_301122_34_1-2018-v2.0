Introduccion al SQL

Estimado (a) estudiante, le damos una grata la bienvenida al Objeto Virtual de Información (OVI), donde se abordaran los temas propuestos para el curso administración de base de datos, donde le daremos una breve introducción y las generalidades, esperamos que el curso y los temas contenidos en este sean de su agrado.

A continuación daremos una breve introducción al modelo relacional desglosaremos algunos conocimientos básicos que le ayudaran a comprender mejor este amplio tema también podremos a su disposición referencias para que el tema sea entendible y ameno.

El SQL es un lenguaje de consulta estructurada, es simple e interactivo que permite el acceso a bases de datos relacionales donde busca declarar distintos tipos de instrucciones en ellas, basándose en la utilización de las matemáticas y de cálculos relacionales, el SQL tiene la posibilidad de realizar consultas con el objetivo de recuperar información de las bases de datos de manera sencilla. Las consultas se hacen por medio de lenguaje de comandos que permite seleccionar, insertar, actualizar, averiguar la ubicación de los datos, entre muchos otros.

Los responsables de publicar este lenguaje, tambien publicaron el ANSI y la ISO, por lo expuesto anteriormente se puede encontrar este lenguaje en cualquier de los DBMS

En el trascurso del curso veremos los comando más útiles y comunes, como por ejemplo INSERT se utiliza cuando quieres añadir o insertar nuevos datos. UPDATE se utiliza cuando quieres cambiar o actualizar datos existentes. DELETE se utiliza cuando quieres eliminar o borrar datos existentes, entre muchos otros que seran muy utiles para este curso.
